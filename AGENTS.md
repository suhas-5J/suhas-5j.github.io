# Portfolio Project: The Knowledge Orbit

## Project Overview
A high-performance, visually appealing portfolio built with Astro, designed for GitHub Pages. It features a unique 1st-person "Interstellar Odyssey" narrative.

## Tech Stack
- **Framework:** Astro (Content Layer API)
- **Styling:** Tailwind CSS 4.0
- **Interactivity:** React + Framer Motion (Physics-based)
- **Content:** MDX
- **Audio:** Localized Cinematic Engine
- **Icons:** Lucide React

## Core Features
1. **Beyond The Numbers (Hero):** Atmospheric introduction to the dual expertise (Finance + Education).
2. **The Interstellar Journey:** A 1st-person ship-view experience diving through meteorites toward a sequence of planets (Mars ➔ Sun).
3. **The Research Lab:** Unified projects section for financial models and technical showcases.
4. **System Logs (The Journal):** MDX-powered insights on health, society, and growth.
5. **Cockpit HUD:** Holographic interface with Nitrous Warp Jumps, Mute Toggles, and real-time Telemetry.
6. **Snap-Scroll Engine:** Premium, weighted physics navigation for a curated presentation.
