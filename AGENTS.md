# Portfolio Project: The Knowledge Orbit

## Project Overview
A high-performance, visually appealing portfolio built with Astro, designed for GitHub Pages.

## Tech Stack
- **Framework:** Astro
- **Styling:** Tailwind CSS
- **Interactivity:** React + Framer Motion
- **Content:** MDX
- **Icons:** Lucide React

## Core Features
1. **The Launch (Hero):** Introduction to the dual expertise (Finance + Education).
2. **The Journey (Experience):** Scroll-linked rocket animation traversing a "Solar System" of career milestones.
3. **The Lab (Finance/Tech):** Interactive financial models and technical skill showcases.
4. **The Journal (Education/Blog):** MDX-powered insights on health, social issues, and more.
